# big-data-course

## Setup

- install/run:
    `./run-cluster.sh`
- create topic in Kafka:
    `./task.sh`

- run producer:
    `./task.sh producer`
-run consumer:
    `./task.sh consumer`

- delete/stop
    `./shutdown-cluster.sh`
